# BigData-Final-Project-2026


유기동물 발생 패턴 및 입양 영향 요인 분석 프로젝트
1. 문제 정의
   주제: 공공데이터를 활용한 전국 유기동물 발생 현황 및 지역별/품종별 입양 특성 분석
   배경: 매년 유기동물 발생 건수가 증가함에 따라 보호소의 운영 부담이 커지고 있습니다. 데이터 기반의 분석을 통해 유기동물의
   보호 기간을 단축시키고 입양률을 높일 수 있는 인사이트를 도출하고자 합니다.
2. 데이터 수집
   데이터 소스: 공공데이터포털 - 농림축산검역본부 유기동물 보호 현황 API
   수집 방법: Python 스크립트를 이용한 REST API 호출 및 CSV 파일 저장 자동화
3. 기술 스택
   Ingest: Python
   Storage: HDFS
   Processing: Apache Spark
   Visualization: Matplotlib, Seaborn
4. 분석 내용
   1. 휴가철이나 명절 기간에 유기동물 발생 건수가 실제로 급증하는가?
   2. 각 시도별 인구 대비 유기동물 발생률과 입양 성공률 사이의 상관관계?
   3. 동물의 축종, 품종, 연령이 보호 기간 및 입양 여부에 미치는 영향은 무엇인가?
5. 구현 계획
   11주차: Repository 생성 및 문제 정의 완성
   12주차: API 수집 스크립트 작성 및 100MB 이상의 raw 데이터 확보
   13주차: HDFS 적재 및 Spark/Hive를 활용한 파이프라인 구현
   14주차: 데이터 시각화 및 최종 분석 보고서 작성
