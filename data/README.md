# 📂 Data Directory Overview

이 디렉토리는 농림축산식품부 검역본부 Open API로부터 수집된 원본(Raw) 유기동물 데이터가 저장되는 공간입니다. 
과제 요구사항(100MB 이상 대용량)에 따른 원본 CSV 파일은 `.gitignore` 처리되어 GitHub 레포지토리에는 업로드되지 않으며, 메인 파이프라인(`run_all.sh`) 가동 시 자동으로 생성 및 적재됩니다.

### 📍 1. 데이터 출처 (Data Source)
* **제공 기관**: 농림축산식품부 농림축산검역본부 (국가동물보호정보시스템)
* **데이터셋명**: 유기동물 정보 조회 서비스 Open API
* **수집 대상**: 2024년 ~ 2026년 전국 유기동물 공고 데이터 전체

### 📋 2. 원본 데이터 전체 스키마 (Full Data Schema)
API를 통해 수집되어 `raw_animals.csv`에 적재되는 전체 22개 컬럼 명세입니다.

| 분류 | 컬럼명 | 데이터 타입 | 설명 | 비고 (분석 및 활용) |
| :--- | :--- | :--- | :--- | :--- |
| **💡 핵심 식별자** | `desertionNo` | String | 유기번호 | 데이터 고유 식별자 (Primary Key) |
| **📅 일자 및 장소**| `happenDt` | String | 발견일자 (YYYYMMDD) | 시계열 타임라인 분석 (발견 월 추출) |
| | `happenPlace` | String | 발견장소 | 유기 빈번 지역 매핑 활용 |
| **🐕 동물 정보** | `kindNm` | String | 품종 정보 (예: [개] 믹스견) | 축종(개/고양이) 및 품종별 통계 분석 |
| | `colorNm` | String | 모색 (털 색상) | 유기동물 외형적 특징 데이터 |
| | `age` | String | 나이 (예: 2024(년생)) | 문자열 파싱 후 정수형 변환 (연령대 분석) |
| | `weight` | String | 체중 (예: 3.5(Kg)) | 문자열 파싱 후 실수형 변환 (체중별 분석) |
| | `sexCd` | String | 성별 (M: 남, F: 여, Q: 미상) | 성별에 따른 입양/안락사 상관관계 분석 |
| | `neuterYn` | String | 중성화 여부 (Y, N, U: 미상) | 중성화 여부와 유기/입양률 상관분석 |
| | `specialMark` | String | 특징 및 특이사항 (텍스트) | 키워드 매칭을 통한 특이사항별 통계 분석 |
| **📢 공고 및 상태**| `noticeNo` | String | 공고번호 | 관할 기관별 공고 관리 번호 |
| | `noticeSdt` | String | 공고시작일 (YYYYMMDD) | 법적 공고 기한 및 보호소 체류일 계산 |
| | `noticeEdt` | String | 공고종료일 (YYYYMMDD) | 법적 공고 기한 및 보호소 체류일 계산 |
| | `processState`| String | 보호상태 (입양, 안락사 등)| 최종 분석의 핵심 종속 변수 (결과 산출) |
| **🏥 보호소 정보** | `careNm` | String | 보호소명 | 보호소별 관리 규모 및 유기동물 수용량 |
| | `careTel` | String | 보호소 전화번호 | 메타데이터 (분석 제외) |
| | `careAddr` | String | 보호소 주소 | 지자체 위치 매핑 데이터 |
| **🏢 관할 기관** | `orgNm` | String | 관할 지자체명 | 지역별 유기동물 발생 및 처리율 분석 |
| | `chargeNm` | String | 담당자 이름 | 메타데이터 (분석 제외) |
| | `officetel` | String | 담당자 연락처 | 메타데이터 (분석 제외) |
| **🖼️ 이미지 인프라**| `filename` | String | 썸네일 이미지 경로 | 메타데이터 (분석 제외) |
| | `popfile` | String | 원본 이미지 경로 | 메타데이터 (분석 제외) |
